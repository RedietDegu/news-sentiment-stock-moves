## Notebooks

This folder contains the main analysis notebooks for the project:

- `task1_eda_news.ipynb` – Exploratory Data Analysis of the financial news dataset (headline lengths, publishers, time patterns, basic keyword exploration).
- `task2_tech_indicators.ipynb` – Technical indicators (MA, RSI, MACD, etc.) computed on stock price data and visualized.
- `task3_sentiment_correlation.ipynb` – Sentiment analysis of headlines and correlation with daily stock returns.

Run these notebooks in order, using the same Python environment defined in `requirements.txt`.

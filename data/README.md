## Data directory

Place all datasets used in this project under this `data/` folder.

Suggested layout:

- `data/raw/` – original input files (read-only).
  - Example: `fns_news.csv` (FNSPID news dataset).
  - Example: `AAPL_prices.csv` (daily OHLCV for AAPL).
- `data/interim/` – intermediate processed outputs.
- `data/processed/` – final modelling-ready datasets.

Update the paths in the notebooks if your filenames differ from these examples.
